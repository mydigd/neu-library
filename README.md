# 📚 NEU Library Visitor Log System
 
> A full-stack web application for managing and monitoring library visitors at New Era University.
 
🌐 **Live Application:** [https://neulibraryloginsystem.netlify.app](https://neulibraryloginsystem.netlify.app)

## 🏛️ Overview
 
The **NEU Library Visitor Log System** is a web-based application designed to streamline the process of logging and monitoring library visits at New Era University. It replaces manual logbooks with a digital system that provides real-time visitor statistics, role-based access control, and a clean, modern interface.
 
Visitors sign in using their institutional Google account (`@neu.edu.ph`), while admins have access to a full dashboard with visitor analytics, logs management, and account controls.
 
---

## ✨ Features
 
### 👤 Visitor
- Sign in with **NEU Google Account** (`@neu.edu.ph` only)
- Auto-generated **Student Number** on first login (e.g. `NEU-2025-00123`)
- First-time setup: select **College** and **Employee Status**
- Log visit by selecting **Purpose of Visit**
- Recognized automatically on return visits
 
### 🛡️ Admin
- Sign in with authorized **Google Account**
- **Dashboard** — visitor statistics by day, week, month, or custom date range
- **Visit Logs** — view and delete individual visit records
- **Visitors** — add, edit, and remove registered visitor accounts
- **Admins** — manage admin accounts (add/remove)
- Filter statistics by **purpose**, **college**, and **visitor type**
- Live **search** across all visitor records
